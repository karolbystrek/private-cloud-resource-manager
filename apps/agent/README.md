# Private Cloud Resource Manager - Agent

This directory contains the Python-based agent that runs on worker nodes (and the "Virtual Node").

## Setup

1.  Create a virtual environment:
    ```bash
    python3 -m venv venv
    source venv/bin/activate
    ```
2.  Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```
3.  Run the agent:
    ```bash
    python main.py
    ```
